    What is Web Cards UI?

    Web Cards UI is a simple way to create card style UI in HTML using different data sources. Include Jquery and the Included Javascript file. Developers then simply have to Initiate the Web Cards with a simple Javascript call defining the data source along with cards container. The cards are automatically generated.

    Why is Web Cards UI Important?

    As the web moves to mobile devices, there is a paradigm shift on the focus of content vs. presentation. There are a large number of major content providing websites that do not offer a simplified way to read their stories or articles, We are going to change that.
